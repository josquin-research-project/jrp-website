Keyboard Shortcuts

Various pages on the JRP website have keyboard shortcuts. This page lists the available shortcuts for each page type.

Browse page

shift+right-arrow
View next page of the browsing list.
shift+left-arrow
View previous page of the browsing list.
control+minus
Decrease the number of lines in the browse table.
control+equals
Increase the number of lines in the browse table.
control+A
Toggle display of browse table between paged and unpaged views.
control+J
Toggle display of catalog number/genre in first column of browse table.
control+n
Show/hide number of works in the browse list. The work count is shown underneath the browse form.

Work page

shift+right-arrow
Go to next work page.
shift+left-arrow
Go to previous work page.
control+shift+right-arrow
Go to first work in next repertory.
control+shift+left-arrow
Go to first work in previous repertory.
shift+home
Go to first work in first repertory.
shift+end
Go to last work in last repertory.
shift+up-arrow
Go to previous section in a multi-section work.
shift+down-arrow
Go to next section in a multi-section work.
1 — 9 / control+1 — 9
Display speicified section number. Control-number will work with any focus.
I / control+I
Toggle view of all incipits for multi-section works. R-key will work when not in an input box. Control-R will work with any focus.
P / control+P
Play audio for currently selected section (or work for single-sectioned work). P-key will work when not in an input box. Control-P will work with any focus.
R / control+R
Toggle view of all vocal ranges for multi-section works. R-key will work when not in an input box. Control-R will work with any focus.
S / control+S
Toggle table view of sections (for multi-sectioned works). S-key will work when not in an input box. Control-S will work with any focus.
D / control+D
Toggle range type view (histograms by note counts or duration weighted). D-key will work when not in an input box. Control-D will work with any focus.
space
Toggle play/pause audio button of currently active audio recording. Not active when focus is on a input box (use shift-space in that case).
shift+space
Toggle play/pause audio button of currently active audio recording.
