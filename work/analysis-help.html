JRP Analysis Tools

Activity. These plots show:

  1. the total number of note attacks per measure, taking all voices into account
  2. the relative activity of each voice (black = very active; white = rest)

Parallel. Displays all parallel unisons, octaves, and fifths (diatonic) in a given repertory. The JRP distinguishes between “hard” parallel motion, where the initial notes are attacked simultaneously, and “soft” parallel motion, where they are staggered.

Rhythm. Inspired by Sean Gallagher’s important article on style and syntax in the music of Johannes Ockeghem, this tool displays every measure-long rhythmic pattern—that is, every pattern that starts at the beginning of a measure and lasts exactly one breve (or, under some mensuration signs, one long). Patterns are listed in order of frequency; click any gesture for the passages in which it appears.

Range. Displays the total range of each voice. Also shows the relative frequency of each pitch: (1) based on note attacks; (2) based on duration.

An additional feature is available on all Work pages: mouse over the range graph to see the precise number of times a given note is sung.
